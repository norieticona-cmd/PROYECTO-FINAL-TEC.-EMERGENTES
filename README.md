# PROYECTO-FINAL-TEC.-EMERGENTES

## 🏆 Hackathon Manager

Proyecto final de la materia de Tecnologías Emergentes realizado por el equipo "El Dashboard".

Sistema Web de Gestión de Hackathons desarrollado con Flask.

## 📋 Descripción

Hackathon Manager es una aplicación web que permite administrar eventos Hackathon, gestionar equipos, participantes, mentores, proyectos, evaluaciones y premios.

El sistema facilita el control integral de competencias tecnológicas mediante una plataforma centralizada.

## 🚀 Tecnologías Utilizadas

* Python 3
* Flask
* SQLAlchemy
* SQLite
* Flask-Login
* Bootstrap 5
* HTML5
* CSS3

## 📊 Funcionalidades

### Seguridad

* Registro de usuarios
* Inicio de sesión
* Gestión de roles

### Gestión

* Hackathons
* Equipos
* Participantes
* Mentores
* Proyectos
* Evaluaciones
* Premios

### Dashboard

* Estadísticas generales
* Proyecto líder
* Promedio de evaluaciones
* Último premio registrado
